# Smart Library Book Reservation System

A Java-based library management system that supports fast book search, priority-based reservations, and demand analysis using core data structures.

## Features
- Fast prefix-based book search using Trie
- Priority-based reservation handling
- Demand tracking for popular books
- Modular and scalable design

## Data Structures Used
- Trie
- Priority Queue (Heap)
- HashMap

## Tech Stack
- Java
- Core DSA concepts
